# Roles

[Framework README](../README.md) | [Role index](../docs/roles/README.md)

Roles are grouped by operational scope:

- `INFRA/` configures shared FlexPod infrastructure.
- `TENANT/` configures one selected tenant.
- `rancher/` configures RKE2.
- `ONTAP/` contains shared or legacy ONTAP building blocks.
- Other top-level roles are legacy or specialized helpers and should be validated before use.

Open the README in each role directory for purpose, task flow, functions called, inputs, expected configuration, and product references.
